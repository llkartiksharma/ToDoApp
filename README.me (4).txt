📝 Java Swing To-Do List App

📌 Description
This is a simple To-Do List desktop application built using Java Swing. The app allows users to add and delete tasks with a clean graphical user interface using components like JFrame, JButton, JTextField, and JList.

🚀 Features
✅ Add new tasks to the list

🗑 Delete selected tasks

🖥 Simple and intuitive GUI using Swing

💡 Real-time interaction without reloading

🛠 Requirements
Java 8 or higher

A Java IDE (like IntelliJ, Eclipse) or command-line tools

📂 How to Run

1. Save the File
Save the code as:
ToDoApp.java

2. Compile the Code
javac ToDoApp.java

3. Run the Application
java ToDoApp


🧑‍💻 Usage Instructions
Type a task into the input box at the bottom.

Click Add Task to add it to your to-do list.

Click on a task in the list to select it.

Click Delete Selected to remove the selected task.
